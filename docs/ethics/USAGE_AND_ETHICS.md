# Usage & Ethics Statement

This system is designed as a **research assistant**, not an authorship substitute.

## Allowed
- Quoting public code for analysis
- Summarizing patterns and approaches
- Highlighting license obligations

## Not Allowed
- Presenting external code as original output
- Silent derivation of implementations
- Automatic incorporation into production systems

## Guiding Rule
> You may quote and summarize public code for research.
> You must not present it as original work or generate derivative implementations without explicit transformation and explanation.

This aligns with academic research norms, open-source licenses, and responsible AI use.
